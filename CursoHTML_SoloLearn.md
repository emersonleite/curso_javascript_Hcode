## A tag <html>
  
Embora várias versões tenham sido lançadas ao longo dos anos, os fundamentos do HTML permanecem os mesmos.  
  
A estrutura de um documento HTML foi comparada com a de um sanduíche. Como um sanduíche tem duas fatias de pão, o documento HTML tem abertura e fechamento de tags HTML.  
  
Essas tags, como o pão em um sanduíche, envolvem todo o resto:

```html
<html>  
…</html>
```

## A tag \<head>
  
  
Imediatamente após a tag HTML de abertura, você encontrará o **cabeçalho** do documento, que é identificado pela abertura e fechamento das tags head.  
  
O cabeçalho de um arquivo HTML contém todos os **elementos não visuais** que ajudam a fazer a página funcionar.

```html
<html> <head> … </head></html>  
```
Os elementos da seção principal serão discutidos posteriormente.

## A tag \<body>

  
  
A tag **body** segue a tag head.  
Todos os elementos estruturais visuais estão contidos na tag body.  
  
Títulos, parágrafos, listas, citações, imagens e links são apenas alguns dos elementos que podem estar contidos na tag body.  
  
**Estrutura HTML básica:**

```html
<html>  
<head>  
</head> <body> </body>** </html>
```

## A tag \<title>

  
  
Para colocar um título na guia descrevendo a página da web, adicione um elemento \<title> à sua seção head:
```html
<html>  
<head> 
<title> first page </title>  
</head>  
<body>  
Esta é uma linha de texto.  
</body></html>
```
